Добро пожаловать в БД Treehouse!

БД будет хранить информацию по обучающей платформе для инженеров программного обеспечения.
Основная таблица в БД - таблица с курсами такими как HTTP_Basics или Object-Oriented_Python, от нее зависит таблица с шагами в курсе.
По мере выполнения шагов в таблице достижений будут добаляться записи о присвоении значка за прохождение шага у соответствующего пользователя.
С помощью БД можно отследить информацию по пользователю, его данные в профиле, выбранную подписку и историю платежей.
Отдельно будут сформированы таблицы с запланированными для прохождения и пройденными курсами.
Будет создана таблица с курсами по направлению обучения, будь то Full_Stack_JavaScript или Beginning_Python.
Присутствует таблица с уведомлениями пользователя и инструкторами курсов.

Вот некоторые задачи, решаемые БД:

* подсчитать количество пройденных пользователем курсов;
* подсчитать общее заработанное количество очков;
* просмотреть историю оплат обучения;
* просмотреть какие курсы необходимо выполнить для выбранного направления

БД содержит 13 таблиц:

Courses, Steps, Achievements, Bookmarks, Completed_courses, Tracks, Topics, Users, Profiles, Subscription, Subcription_plans, Billing_history, Instructors


Для создания ERD использовал MySQL Workbench.
Помимо файлов с расширением mwb приложу скриншот схемы.
